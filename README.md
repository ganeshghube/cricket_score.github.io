# 🏏 Cricket Match Streaming & Score Overlay (Free Club Use)

A **free-to-use cricket scoring overlay system** designed for **club matches, local tournaments, and practice games**.
This project allows you to **update live scores from an admin panel** and **overlay them on live YouTube streams**.

---

## 🎯 Features

* ✅ Live cricket scoring (manual)
* ✅ Professional score overlay
* ✅ YouTube streaming compatible
* ✅ Works with OBS / FFmpeg
* ✅ Admin panel for score updates
* ✅ Free for club cricket matches
* ✅ Lightweight & browser-based
* ✅ No installation required

---

## 🖥️ How It Works

1. Open **Admin Page**
2. Enter:

   * Team Names
   * Players
   * Score
   * Overs
   * Wickets
3. Overlay page automatically updates
4. Add overlay page to **OBS / YouTube stream**
5. Start streaming

---

## 🎥 How to Use with OBS Studio

### Step 1 — Open OBS

Add a new **Browser Source**

### Step 2 — Paste Overlay URL

```
https://ganeshghube.github.io/cricket_score.github.io/index.html
```

### Step 3 — Set Resolution

```
Width: 1920
Height: 1080
```

### Step 4 — Position Overlay

Place it above your camera/video layer.

### Step 5 — Start Streaming

Score updates from admin page will appear on stream.
